# n8n 다른 PC 이전 및 설치 가이드

이 문서는 현재 PC의 n8n 환경을 다른 PC로 그대로 복사하여 설치하고 운영하는 방법을 설명합니다.

---

## 1. 사전 준비 (새 PC)
n8n 폴더를 복사하기 전에 새 PC에 다음 환경이 갖춰져 있어야 합니다.

1. **Node.js 설치**: [Node.js 공식 홈페이지](https://nodejs.org/)에서 **v22** 이상의 LTS 버전을 설치합니다.
2. **PM2 설치**: 터미널(CMD 또는 PowerShell)을 관리자 권한으로 열고 아래 명령어를 입력합니다.
   ```bash
   npm install -g pm2
   ```

---

## 2. 파일 복사 및 이동
1. **프로그램 폴더**: 기존 PC의 `C:\n8n` 폴더 전체를 새 PC의 `C:\n8n` 경로에 그대로 복사합니다.
   - *주의: 경로가 달라지면 GUI 도구(`run-pm2-manager.vbs`)가 작동하지 않을 수 있으므로 가급적 동일한 경로를 권장합니다.*

2. **데이터 폴더 (선택 사항)**: 기존에 작성한 워크플로우와 설정을 그대로 옮기려면 아래 폴더를 복사해야 합니다.
   - **경로**: `C:\Users\<사용자이름>\.n8n`
   - 위 폴더를 새 PC의 `C:\Users\<새사용자이름>\.n8n` 위치에 복사합니다.

---

## 3. 서비스 등록 및 실행
파일 복사가 완료되면 새 PC에서 n8n 서비스를 PM2에 등록해야 합니다.

1. **터미널 실행**: 관리자 권한으로 터미널을 엽니다.
2. **n8n 실행 등록**:
   ```powershell
   cd C:\n8n\n8n-instance
   pm2 start node_modules/n8n/bin/n8n --name n8n-instance
   ```
3. **정상 작동 확인**: 브라우저에서 [http://localhost:5678](http://localhost:5678) 접속이 되는지 확인합니다.

---

## 4. 자동 실행 설정 (GUI 도구 활용)
새 PC에서도 재부팅 시 n8n이 자동으로 켜지도록 설정합니다.

1. `C:\n8n` 폴더의 **`run-pm2-manager.vbs`** 파일을 더블 클릭합니다.
2. **[자동 실행 등록 (Register)]** 버튼을 클릭합니다.
3. 완료 메시지가 뜨면 이제 새 PC에서도 재부팅 시 n8n이 자동으로 시작됩니다.

---

## 5. 체크리스트
- [ ] 새 PC에 Node.js v22+ 가 설치되었는가?
- [ ] `npm install -g pm2` 명령어로 PM2를 설치했는가?
- [ ] `C:\n8n` 경로에 파일이 모두 복사되었는가?
- [ ] `pm2 start` 명령어로 프로세스를 최초 등록했는가?
- [ ] GUI 도구로 자동 실행(Startup) 등록을 마쳤는가?
